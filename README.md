# Frontend-App

## Description

Frontend-App is a web application built for demonstrating the capabilities of a front-end developer. It showcases a simple yet functional user interface and interaction with a robust backend system. The app allows users to create and manage to-do lists, making it a great example of how to integrate frontend with backend technologies.

## Features

*   User Registration and Login
*   To-Do List management
*   Task creation, editing, and deletion
*   Task categorization
*   Smooth user interface
*   Responsive design for mobile and desktop devices

## Technologies Used

*   Frontend:
    *   React
    *   Redux
    *   React Router
    *   JavaScript ES6
    *   Webpack
    *   Babel
    *   CSS - Styled Components
*   Backend:
    *   Node.js
    *   Express
    *   MongoDB
    *   Mongoose
*   Tools:
    *   Webpack-dev-server
    *   ESLint
    *   Prettier

## Installation

### Prerequisites

*   Node.js (version 14 or higher)
*   npm (version 6 or higher)

### Setup

1.  Clone the repository using git clone `https://github.com/your-username/frontend-app.git`
2.  Install dependencies using `npm install` or `yarn install`
3.  Start the application using `npm start` or `yarn start`

### Running the Application

*   Open `http://localhost:3000` in your web browser to view the application
*   Use the user interface to create, manage, and interact with to-do lists
